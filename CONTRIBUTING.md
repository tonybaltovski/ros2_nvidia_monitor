# Contributing

Thanks for your interest in improving `ros2_nvidia_monitor`.

## Reporting issues

Please open a GitHub issue with:

- A description of the problem or feature request.
- Your ROS 2 distribution, OS, NVIDIA driver version, and GPU model
  (output of `nvidia-smi` is ideal).
- Steps to reproduce, if applicable.

## Pull requests

- Keep changes focused; one logical change per PR.
- Run `pre-commit run --all-files` before pushing.
- Ensure CI is green on all supported ROS 2 distributions.
